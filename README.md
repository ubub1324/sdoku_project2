스도쿠 게임
스도쿠는 가로, 세로 그리고 3x3의 작은 정사각형 안에서
1부터 9까지의 숫자를 중복되지 않게 채워 넣는 논리 퍼즐입니다.

규칙
각 행에는 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.
각 열에는 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.
각 3x3 영역에도 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.

파일 설명
initialize_board.py : 빈 스도쿠 보드를 생성합니다.
print_board.py : 현재 스도쿠 보드를 출력합니다.
solve_sudoku.py : 스도쿠 퍼즐을 해결합니다.
generate_sudoku.py : 문제를 생성합니다.
play_sudoku.py : 사용자가 직접 플레이할 수 있습니다.
